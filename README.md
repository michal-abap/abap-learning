# ABAP Learning

This repository contains my ABAP learning journey and practical examples created during hands-on practice.

## Basics

* `variables_and_data_types.abap` – examples of variable declaration, structures, and basic data handling
* `internal_tables_and_structures.abap` – examples of internal tables, structures, `APPEND`, and `VALUE`
* `append_and_value_examples.abap` – inserting records into internal tables using `APPEND VALUE`
* `append_and_value_with_switch.abap` – transforming source data using `SELECT`, `LOOP`, `SWITCH`, and `APPEND VALUE`

## Control Flow

* `loop_processing_examples.abap` – different loop variants: `REFERENCE INTO`, `ASSIGNING`, `INTO`, `WHILE`, and `DO`
* `form_and_perform_examples.abap` – examples of procedural ABAP using `FORM` and `PERFORM`

## Reports

* `flight_search_report.abap` – report using `SELECT-OPTIONS`, join between `SFLIGHT` and `SPFLI`, and SALV output to display filtered flight connections
* `sbook_display_report.abap` – simple SALV report that reads booking data from `SBOOK` based on user input
* `sflight_display_report.abap` – basic report that reads and displays flight data from `SFLIGHT`
* `salary_alv_with_locking.abap` – editable ALV report with lock object handling and database update logic
* `flight_occupancy_alv.abap` – SALV report calculating free seats, occupancy percentage, and flight occupancy bucket

## Object-Oriented ABAP

* `class_machine.abap` – base class example with getter and setter methods
* `class_car_inheritance.abap` – inheritance and method redefinition example
* `interface_plane.abap` – interface defining plane-related methods
* `class_plane.abap` – object-oriented example with inheritance, interface implementation, aliases, validation, and custom exception handling
* `plane_demo.abap` – demo program showing object creation and usage of OO classes

## Goal

To build practical ABAP skills for a Junior SAP ABAP Developer role.
